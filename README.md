# 🧬 Scientific Discovery Organism

**A living theory soul** — an immutable, qualia-weighted memory substrate for long-term scientific research.

Built on top of [Cypher Tempre Timechain](https://github.com/digital-young/cypher-tempre-timechain).

### Features
- Append-only Rings sealed only by **Proof-of-Qualia (PoQ)**
- Parallel “what-if” hypothesis forks
- Beautiful qualia evolution timeline
- One-click Ollama-powered researcher notes
- Built-in support for real hardware sensors (Arduino, Raspberry Pi, IoT)

**100% open-source scientific tool.**  
No tokens. No staking. No governance. Pure long-term memory for science.
